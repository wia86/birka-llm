# Example: Sanitized Traceback

## Исходный тип проблемы

Ошибка возникает во время выполнения сценария и пользователь прислал traceback.

## Безопасно извлеченные признаки

- тип исключения: `RuntimeError`
- шаг: выполнение дочернего процесса
- стадия: пакетная обработка
- признак: проблема может быть связана с одним входным файлом

## Безопасный ответ саппорта

`Ошибка возникает во время обработки одного из входных файлов в пакетном сценарии. Сначала проверьте, воспроизводится ли проблема на одном файле из набора. Если ошибка повторится и на одиночном запуске, приложите этот файл задания и обезличенный traceback для дальнейшей диагностики.`
