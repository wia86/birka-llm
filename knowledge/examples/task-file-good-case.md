# Example: Good Task File Case

## Сценарий

Пользователь прислал файл задания, который корректно относится к модулю `calc`.

## Безопасно извлеченные признаки

- модуль: `calc`
- тип источника: папка
- есть панели исходных данных
- есть как минимум одна панель результата
- нет явных признаков старого формата

## Ответ саппорта

`Файл задания выглядит корректно: это сценарий модуля calc с заданным источником данных и настроенными панелями результата. Если проблема возникает при запуске, следующим шагом нужен traceback или текст итоговой ошибки.`
