# Example: Task File Panel Conflict

## Сценарий

В задании включены конфликтующие или дублирующиеся панели результата.

## Безопасно извлеченные признаки

- модуль: `calc`
- найдено более одной включенной панели одного и того же результата

## Вероятная причина

Пользователь дублировал панель результата и сохранил задание в таком виде.

## Ответ саппорта

`В задании есть повтор одной и той же панели результата. Откройте задание, оставьте включенной только одну копию нужной панели, а остальные отключите или удалите, затем повторите запуск.`
