# Example: Missing Source In Task File

## Сценарий

Файл задания загружается, но в нем нет корректного источника РМ.

## Безопасно извлеченные признаки

- модуль определен
- файл задания читается
- `source` пустой или не содержит активного пути

## Вероятная причина

Задание было сохранено без источника или перенесено на другую машину, где старый путь больше неактуален.

## Ответ саппорта

`Файл задания удалось прочитать, но в нем не задан или недоступен источник данных. Откройте это задание в соответствующем модуле, заново выберите файл или папку в блоке "Выбор моделей" и повторите запуск.`
