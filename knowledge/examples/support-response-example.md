# Example: Support Response

## Вход

Пользователь сообщает: "После загрузки задания и запуска расчет не стартует, пишет, что не выбраны контролируемые элементы."

## Безопасный ответ

`Похоже, сценарий остановился на этапе проверки задания перед запуском. Обычно это означает, что для выбранных результатов не заданы контролируемые элементы. Откройте панель "Контролируемые элементы" и убедитесь, что включен хотя бы один способ выбора: по полю control или по выборке. После этого повторите запуск.`
