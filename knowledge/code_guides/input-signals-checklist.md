# Input Signals Checklist

Чеклист диагностических сигналов для triage.

## Если пользователь прислал traceback

Выделить:
- тип исключения;
- короткое сообщение;
- модуль или шаг сценария;
- последние 5-10 кадров;
- воспроизводится ли ошибка повторно.

## Если пользователь прислал файл задания

Выделить:
- модуль;
- тип источника;
- состав панелей;
- есть ли результатные панели;
- есть ли признаки старого формата;
- есть ли пустые обязательные секции.

## Если пользователь прислал только текстовое описание

Уточнить:
- какой модуль открыт;
- что именно он пытался сделать;
- на каком шаге произошла ошибка;
- есть ли файл задания;
- есть ли traceback или текст сообщения.

## Если пользователь прислал скрин

Уточнить:
- текст ошибки в виде копируемого текста;
- какой был последний шаг;
- какой файл или модуль использовался;
- можно ли прислать файл задания.
