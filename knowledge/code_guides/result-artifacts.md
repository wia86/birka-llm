# Result Artifacts

Справочник выходных артефактов, которые может получить пользователь после выполнения сценария.

## Основные типы результатов

### Таблицы
- сводные таблицы;
- табличные результаты расчета;
- таблицы по ТКЗ;
- аналитические выборки.

### Графические материалы
- рисунки;
- RG2 и связанные графические файлы;
- материалы для последующей подготовки документов.

### Документы
- пояснительные записки;
- PDF;
- Word-документы;
- сопутствующие отчеты.

### Измененные или преобразованные файлы
- результаты редактирования РМ;
- конвертированные форматы;
- пакетные выходные файлы.

## Как интерпретировать результат в саппорте

### Если результат есть, но пользователь считает его неправильным
- уточнить, какой именно тип результата ожидался;
- сверить состав включенных панелей;
- проверить, не отключены ли важные блоки результата.

### Если часть результатов отсутствует
- проверить, были ли включены нужные панели;
- проверить, не зависит ли результат от внешнего компонента;
- проверить, нет ли предупреждений в итоговых сообщениях.

### Если создана только часть файлов
- определить, какая часть сценария завершилась успешно;
- уточнить, был ли расчет общий или сервисный;
- проверить traceback и диагностические сообщения.
