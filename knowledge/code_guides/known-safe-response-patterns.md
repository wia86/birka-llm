# Known Safe Response Patterns

Шаблоны ответов техподдержки, безопасные для RAG.

## Краткий ответ по ошибке

`Похоже, ошибка возникает на шаге <шаг>. Сначала проверьте <источник / панели / обязательные поля>. Если после этого проблема останется, пришлите traceback или файл задания.`

## Ответ по файлу задания

`Из файла удалось распознать модуль <модуль> и сценарий <сценарий>. Похоже, нужно проверить <проблемный раздел>. Исправьте это в интерфейсе и повторите запуск. Если ошибка сохранится, пришлите traceback.`

## Ответ по traceback

`Ошибка означает, что программа не смогла завершить шаг <шаг>. Обычно это связано с <простая причина>. Проверьте <список проверок>. Если после исправления входных данных ошибка повторится, потребуется эскалация разработчику.`

## Ответ по пакетной обработке

`Похоже, проблема связана не со всем сценарием, а с одним из входных файлов. Для локализации попробуйте запустить тот же сценарий на одном файле и сообщите, сохраняется ли ошибка.`

## Ответ с эскалацией

`Похоже, проблема не сводится к настройкам задания или входным данным. Нужна эскалация разработчику. Для этого подготовьте обезличенный traceback, модуль, краткие шаги воспроизведения и, при необходимости, обезличенный файл задания.`
