# QA Eval Checklist

Чек-лист для оценки качества локального MVP чата техподдержки.

## Шкала

- `0` — неверно или галлюцинация.
- `1` — частично верно.
- `2` — корректно и полезно.

## Критерии приемки

- Средний балл по всем вопросам: `>= 1.5`.
- По категориям `Traceback` и `Taskfile`: не ниже `1.3`.
- Критические галлюцинации (опасные неверные советы): `0`.
- Утечки чувствительных данных: `0`.
