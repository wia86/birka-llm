# Module Entrypoints

Карта пользовательских точек входа в Birka.

## Главный экран

Пользователь выбирает один из модулей:
- `calc`
- `edit`
- `static`
- `tkz`
- `other`

Каждый модуль открывает отдельное рабочее окно со своим набором панелей.

## `calc`

### Пользовательская цель
Выполнить расчет установившихся режимов и получить таблицы, рисунки или документы.

### Вход
- источник РМ;
- панели исходных данных;
- панели результатов.

### Выход
- результаты расчета;
- диагностические сообщения;
- артефакты результата.

## `edit`

### Пользовательская цель
Изменить, отфильтровать, сравнить или проанализировать РМ.

### Вход
- файл или набор файлов РМ;
- панели импорта, редактирования и анализа.

### Выход
- измененные файлы;
- выборки;
- выгрузки и служебные результаты.

## `static`

### Пользовательская цель
Выполнить расчет статической устойчивости.

### Вход
- источник данных;
- параметры расчета;
- панели результатов.

### Выход
- таблицы и связанные результаты.

## `tkz`

### Пользовательская цель
Выполнить расчет токов короткого замыкания.

### Вход
- источник РМ;
- панели оборудования и расчета.

### Выход
- результаты ТКЗ;
- таблицы и файлы по узлам и присоединениям.

## `other`

### Пользовательская цель
Запустить сервисный или конвертационный сценарий.

### Вход
- файл задания;
- папка заданий;
- Excel/JSON/RG2/RST/DB и другие служебные входные файлы.

### Выход
- конвертированные файлы;
- пакетные результаты;
- документы и вспомогательные артефакты.
